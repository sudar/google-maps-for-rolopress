=== Google Maps for RoloPress ===
Contributors: rolopress, SlipFire LLC.
Requires at least: WordPress 2.8, RoloPress Core 1.4
Tested up to: WordPress 2.8.1, , RoloPress Core 1.5
Stable tag: 0.1

Display a Google Map from your contact and company address in any widget area of RoloPress.

== Installation ==

1. Upload the google-map-for-rolopress folder to your /wp-content/plugins/ directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.

== Frequently Asked Questions ==

= Can multiple instances of the widget be used? =

Yes.

= Do I need a Google API key? =

Nope.

= Is there a limit to how many maps I can show? =

Google limits usage to 1,000 maps a day.
